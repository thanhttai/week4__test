#1 What is CLI?
1 CLI is a command line program that accepts text input to execute operating system functions.
2 In the 1960s, using only computer terminals, this was the only way to interact with computers.
3 In the 1970s an 1980s, command line input was commonly used by Unix systems and PC systems like MS-DOS and Apple DOS.
#2 What is the advantage/disadvantage of CLI?
Advantages of command line interface (CLI)
1 Fast performance: CLI is an interface in which different text commands are written and then these commands are executed by the operating system. It is a text-based interface. CLI is most popular in the 1960s when there was no GUI (graphical user interface) available for computers.
2 Less memory:During the early days, there was no mouse available for the operating system and users only interact with the system by typing commands in CLI.
Disadvantages of command line interface (CLI)
1 Difficult to remember:It is difficult to remember all the commands used in the CLI. UNIX has more than 100 commands for CLI. Some commands are used with other commands and remembering the sequence of commands is also difficult.
2 Error handling:If you write wrong commands by any typo then it may delete any file or move the file to the wrong location. If you accidentally delete any file then it creates a problem for you and maybe that file contains important information.
#3 Which do you prefer, CLI or GUI (Graphical User Interface)? Why?
I prefer CLI because A command-line interface is user-friendly to a sophisticated user, while a graphical user interface is user-friendly to a newbie, but extremely user-unfriendly to a sophisticated user.

#28 What is vw/vh?

vh & vw. vh stands for viewport height and vw is for viewport width. Hence, setting an element to a width value of 50vw means that the element will have a width that's 50% of the viewport size, and this stays true when the viewport is resized.
